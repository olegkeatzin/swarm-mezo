# Лекция 3. Мультиагентное управление в динамических сетях. Консенсус.

**Основная концепция:** Кооперативное управление (или управление распределенными динамическими системами на графах) описывает ситуации, когда каждый узел сети (агент) может получать информацию для формирования управления только от себя и своих непосредственных соседей. 

*Сложности реальных систем:* неполнота информации, наличие шумов измерения, эффекты квантования (дискретизации) и запаздывания.

---

## 1. Вводный пример: Усреднение в графе
*(на основе первых слайдов и рукописных записей)*

Рассматривается сеть из 5 узлов (агентов).
*   **Начальные состояния узлов ($x^i$):** $x^1=8,\; x^2=22,\; x^3=15,\; x^4=5,\; x^5=0$.
*   **Цель (усреднение):** Достичь среднего значения $\bar{x} = 10$.
*   **Максимальная невязка (ошибка):** $R = \max |x^i - \bar{x}| = 12$ (для узла 2).

> **Аналогия из конспекта:** Данную задачу можно представить как очередь в магазине. Стоит вопрос оптимизации: *"Куда идти покупателю?"*, чтобы сбалансировать нагрузку.

### Итеративное усреднение по соседям (Local Voting Protocol)
Идея похожа на **градиентный спуск**. Каждый узел обновляет свое состояние на основе разницы состояний с соседями.

**Первый подход (адаптивный шаг):**
Формула: $x^i_t = x^i_{t-1} + \alpha^i \sum_{j \in N^i} (x^j_{t-1} - x^i_{t-1})$
Где шаг $\alpha^i = \frac{1}{1 + |N^i|}$ зависит от количества соседей $|N^i|$.
*Пометки из конспекта:* Для узлов 2, 3, 5 коэффициент $\alpha^i = 1/3$, а для узлов 1, 4 коэффициент $\alpha^i = 1/4$. При ручном расчете с такими коэффициентами возникли **ошибки (расходимость/неточность)**.

**Второй подход (фиксированный шаг):**
В конспекте отмечено: *"Пусть везде $\alpha^i = \alpha = 1/4$".* 
С единым фиксированным шагом алгоритм сходится корректно: *"Теперь всё работает хорошо, т.е. мы считываем невязки с фиксированным коэффициентом $\alpha$. Ошибок нет"*.

## 2. Теория графов для динамических сетей

Сеть агентов моделируется с помощью направленного графа (орграфа).
*   $N = \{1, 2, \dots, n\}$ — множество агентов (узлов).
*   $E$ — множество ребер (дуг).
*   $N^i = \{j: (j,i) \in E\}$ — множество **соседей узла $i$** (от кого узел $i$ получает информацию).

**Динамическая топология:**
Структура связей может меняться со временем и описывается последовательностью орграфов $\{(N, E_t)\}_{t \ge 0}$, где текущие связи $E_t \subseteq E$.
> **Пример из конспекта (с людьми):** У человека есть две руки ($E_t = 2$ — текущие активные связи), но в принципе он может взаимодействовать со всеми в аудитории (глобальное множество $E$).

### Матрицы графа
1.  **Матрица смежности $A$:** Определяет веса связей ($a^{ij} > 0$, если дуга существует). Для неориентированного графа она симметрична ($A = A^T$). Из нее можно извлекать вероятности появления ребер.
2.  **Матрица степеней захода $D$:** Диагональная матрица $D = \text{diag}\{d^i\}$, где $d^i$ — in-degree (сколько ребер входит в узел $i$).
3.  **Матрица Лапласа (Лапласиан):** $L(A) = D - A$
    *   *Свойства:* Сумма элементов в каждой строке равна $0$.
    *   Собственный вектор, соответствующий нулевому собственному числу ($\lambda_1 = 0$), состоит из единиц.
    *   **Критерий Гершгорина:** Все собственные числа Лапласиана имеют неотрицательную вещественную часть и лежат в круге радиуса $d_{max}$ с центром в $d_{max}$.
    *   **Второе собственное число ($\lambda_2$)** называется числом Фидлера и играет важную роль в оценке скорости сходимости (алгебраическая связность).
    *   **Лемма об остовном дереве:** Ранг матрицы Лапласа равен $n-1$ тогда и только тогда, когда граф имеет остовное дерево. (Если граф сильно связный $\implies \text{rank}(L) = n-1$).

## 3. Проблема консенсуса

**Динамика состояний узлов:** 
$x^i_{t+1} = x^i_t + f^i(x^i_t, u^i_t)$
где $u^i_t$ — управляющее воздействие.

**Определение консенсуса:**
Узлы $i$ и $j$ синхронизированы в момент $t$, если $x^i_t = x^j_t$. 
Асимптотический консенсус достигается, если существует такое значение $x^*$, что:
$\lim_{t\to\infty} x^i_t = x^* \quad \text{для всех } i \in N$

### Протокол локального голосования (Local Voting Protocol)
Управление формируется как обратная связь по состояниям соседей:
$u^i_t = \alpha \sum_{j \in N^i_t} b^{ij}_t (x^j_t - x^i_t)$
Где $\alpha > 0$ — размер шага управления.

**В матричной форме:**
Обозначим $\bar{x}_t$ — вектор состояний всех агентов. 
Глобальное управление: $\bar{u}_t = -\mathcal{L}(\alpha A) \bar{x}_t$

### Динамика замкнутой системы
**Дискретное время:**
$\bar{x}_{t+1} = (I - \mathcal{L}(\alpha A))\bar{x}_t = P \bar{x}_t$
Где $P = I - \mathcal{L}(\alpha A)$ — **матрица Перрона**.
*Условие сходимости:* Если граф имеет остовное дерево, матрица $P$ имеет одно простое собственное число, равное $1$. Все остальные лежат внутри единичного круга при условии $\alpha < \frac{1}{d_{max}}$. Консенсус достигается, значение $x^*$ зависит от топологии графа.

**Непрерывное время:**
$\dot{\bar{x}}_t = -\mathcal{L}(\alpha A)\bar{x}_t$
Решение: $\bar{x}(t) = e^{-\alpha \mathcal{L}(A)t} \bar{x}(0)$
Взвешенная сумма начальных состояний $\sum z^i x^i_0$ является инвариантом (не зависит от времени) и определяет финальную точку консенсуса $x^*$.

## 4. Наблюдения с шумами и запаздыванием (Реальные условия)

В реальности агенты получают информацию с помехами:
$y^{ij}_t = x^j_{t-d^{ij}_t} + v^{ij}_t$
Где:
*   $v^{ij}_t$ — шум (помеха).
*   $d^{ij}_t \le \bar{d}$ — целочисленное запаздывание (delay).

В таких условиях точный консенсус невозможен. Вводится понятие **Усредненного консенсуса (Mean squared $\varepsilon$-consensus)**.
Система достигает $\varepsilon$-консенсуса, если математическое ожидание квадрата ошибки ограничено:
$E\|x^i_t - x^*\|^2 \le \varepsilon$ для всех $t > T$.
